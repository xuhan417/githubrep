The overtaking simulation main loop is "overtaking_mid_lane.py".
The overtaking uses a different agent, "behavior_agent_overtaking.py". It has customed conditions for freeway overtaking.

ADS regs API related function are written in "ADS_API.py", which read data from ADS regs server.
